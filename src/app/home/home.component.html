Abeeway CoDec Demo

This is a developer tool that demonstrates how to decode uplink messages sent by Abeeway tracker devices and how to encode configuration commands that can be sent to trackers as LoRaWAN downlink messages. The source code of this tool is available at github here.

This tool is developed as an Angular application utilizing the open source abeeway-driver available on guthub too.

Please also note that Abeeway Trackers are integrated with Actility's ThingPark X Location Engine that can be used as a location solver when trackers use Low Power GPS (LP-GPS) and WiFi localisation technologies. If you are interested in testing TPX Location Engine with Abeeway devices please see a demo application server on github.
